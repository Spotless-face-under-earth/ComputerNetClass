# 实验目的

其实这个实验本身和实验一类似，用已经封装好的Socket包进行数据转发就行。只是实验一中传输层协议用的TCP，本次要用UDP，也是在WinSocket库里就写好了的东西。

那我们要做的是什么呢？最基本的UDP协议是无连接且没有重传机制的，因此，我们要做的其实是**给原有的UDP加一个伪首部，通过伪首部里新增的数据结构来实现面向连接、错误检测、丢包重传等任务**。

明白了要做什么，具体实现很简单，就弄一个class，里面包含需要的数据结构，然后在调用sendto()函数前把这个class增加到要发送的内容前面，并且在接收端进行相应分析即可。
